90-second demo path.

A short reviewer flow for trying Kudlit quickly from install to learning support.

  1. Install the Android APK from the v1.0.0 release.
  2. Continue as guest from the branded welcome screen.
  3. Open Scan and show the camera/gallery scanner controls.
  4. Switch to Translate and type a Filipino phrase.
  5. Open Learn to show lessons, glyph references, and quiz-oriented practice.
  6. Switch to Butty for companion explanations and learning guidance.
